Technical Interests

Web Development

I love crafting interactive, responsive websites from the ground up using HTML, CSS, and JavaScript. Building web applications teaches me how to think in terms of user experience, performance, and accessibility—all crucial skills for modern software development.

Recently I’ve been exploring progressive web apps (PWAs) to deliver native-like experiences in the browser, and I’m always experimenting with new CSS techniques like custom properties and Grid Layout.

Data Analysis

Data-driven decision making fascinates me. I enjoy using Python (pandas, NumPy) to clean datasets, analyze trends, and visualize results. Turning raw data into clear insights feels like uncovering hidden stories.

I’m currently learning how to build dashboards with Plotly and Dash, and I’m intrigued by how machine learning models can predict future outcomes from historical data.
