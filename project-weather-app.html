Weather App

Current weather & 5-day forecast for any city

Overview

This Weather App is a responsive web application that allows users to search for current weather conditions and a 5-day forecast by city name. It consumes the OpenWeatherMap API and displays temperature, humidity, wind speed, and weather icons in a clean, user-friendly interface.

Screenshots

Technologies Used

  • HTML5 & CSS3 (Flexbox & Grid)
  • Vanilla JavaScript (Fetch API)
  • OpenWeatherMap REST API
  • LocalStorage for recent searches
